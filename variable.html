JavaScript Variables

In JavaScript, variables are used to store data values. You can declare a variable using the keywords let, const, or var.

Declaring Variables

  • let: Used to declare block-scoped variables.
  • const: Used to declare block-scoped constants that cannot be reassigned.
  • var: Used to declare function-scoped or globally scoped variables (not recommended in modern JavaScript).

Examples


        // Using let
        let name = "John";
        let age = 30;

        // Using const
        const pi = 3.14;    
        console.log(name); // Output: John
        console.log(age); // Output: 30
        
        // Using var
        var city = "New York";
        console.log(name); // Output: John
        console.log(age); // Output: 30
        console.log(pi); // Output: 3.14

        console.log(city); // Output: New York

Variable Naming Rules

  • Variable names must start with a letter, underscore (_), or dollar sign ($).
  • Subsequent characters can be letters, numbers, underscores, or dollar signs.
  • Variable names are case-sensitive (e.g., myVar and myvar are different variables).
  • Avoid using reserved keywords (e.g., if, for, function) as variable names.
