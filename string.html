JavaScript Strings

Strings in JavaScript are used to represent text. They can be created using single quotes, double quotes, or backticks.

Creating Strings

  • Single quotes: 'Hello'
  • Double quotes: "Hello"
  • Backticks (template literals): `Hello`

Examples


        // Using single quotes
        let greeting = 'Hello, World!';

        // Using double quotes
        let farewell = "Goodbye, World!";

        // Using backticks
        let name = 'John';
        let message = `Hello, ${name}!`;

String Methods

  • .length: Returns the length of the string.
  • .toUpperCase(): Converts the string to uppercase.
  • .toLowerCase(): Converts the string to lowercase.
  • .substring(start, end): Extracts a substring from the string.
  • .indexOf(substring): Returns the index of the first occurrence of a substring.
  • .includes(substring): Checks if the string contains a substring.
  • .replace(searchValue, newValue): Replaces a substring with a new value.
  • .trim(): Removes whitespace from both ends of the string.
  • .split(separator): Splits the string into an array of substrings based on a separator.
  • .charAt(index): Returns the character at a specified index.
  • .charCodeAt(index): Returns the Unicode value of the character at a specified index.
  • .startsWith(searchString): Checks if the string starts with a specified substring.
  • .endsWith(searchString): Checks if the string ends with a specified substring.
  • .repeat(count): Returns a new string with the specified number of repetitions of the original string.
  • .localeCompare(compareString): Compares two strings in the current locale.
  • .match(regexp): Matches a string against a regular expression and returns an array of matches.
  • .search(regexp): Searches for a match between a regular expression and the string, returning the index of the first match.
  • .concat(...strings): Combines two or more strings into one.
  • .padStart(targetLength, padString): Pads the current string with another string (the padString) until the resulting string reaches the given targetLength.
  • .padEnd(targetLength, padString): Pads the current string with another string (the padString) until the resulting string reaches the given targetLength.
  • .toString(): Converts the string to a string (useful for converting other data types to strings).
  • .valueOf(): Returns the primitive value of the string object.

Example of String Methods


        let text = 'JavaScript is fun!';
        
        console.log(text.length); // 20
        console.log(text.toUpperCase()); // JAVASCRIPT IS FUN!
        console.log(text.toLowerCase()); // javascript is fun!
        console.log(text.substring(0, 10)); // JavaScript
        console.log(text.indexOf('fun')); // 15
        console.log(text.includes('Java')); // true
        console.log(text.replace('fun', 'awesome')); // JavaScript is awesome!
        console.log(text.trim()); // JavaScript is fun!
        console.log(text.split(' ')); // ['JavaScript', 'is', 'fun!']
        console.log(text.charAt(0)); // J   
        console.log(text.charCodeAt(0)); // 74 (Unicode value of 'J')
        console.log(text.startsWith('Java')); // true
        console.log(text.endsWith('fun!')); // true
        console.log(text.repeat(2)); // JavaScript is fun!JavaScript is fun!
        console.log(text.localeCompare('JavaScript is great!')); // -1 (indicating 'text' comes before the compared string)
        console.log(text.match(/fun/)); // ['fun']
        console.log(text.search(/is/)); // 11
        console.log(text.concat(' Let\'s code!')); // JavaScript is fun! Let's
        console.log(text.padStart(30, '*')); // **********JavaScript is fun!
        console.log(text.padEnd(30, '*')); // JavaScript is fun!**********
        console.log(text.toString()); // JavaScript is fun!
        console.log(text.valueOf()); // JavaScript is fun!
        console.log(text.toString() === text.valueOf()); // true

String Concatenation

Strings can be concatenated using the + operator or template literals.


        let firstName = 'John';
        let lastName = 'Doe';

        // Using + operator
        let fullName = firstName + ' ' + lastName;

        // Using template literals
        let fullNameTemplate = `${firstName} ${lastName}`;

        console.log(fullName); // John Doe
        console.log(fullNameTemplate); // John Doe

Escape Characters

To include special characters in strings, you can use escape characters:

  • \': Single quote
  • \": Double quote
  • \\: Backslash
  • \n: New line
  • \t: Tab

Example of Escape Characters


        let quote = 'She said, "Hello!"';
        let escapedQuote = 'She said, \"Hello!\"';
        let newLineExample = 'First line\nSecond line';

        console.log(quote); // She said, "Hello!"
        console.log(escapedQuote); // She said, "Hello!"
        console.log(newLineExample); // First line
                                    // Second line
